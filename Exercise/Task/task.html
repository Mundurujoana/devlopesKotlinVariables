Exercise

Consider a car. What attributes and properties of a car could we capture using variables and read-only variables? Which attributes would be mutable and which would be immutable?


Create variables and read-only variables for the following attributes of a car, and then print each one.


  1. paintColor
  2. make
  3. model
  4. year
  5. isManualTransmission
  6. seats
  7. mileage
